# Análisis ABC de Activos en Python

## Descripción
Este proyecto implementa un análisis ABC para la clasificación de activos utilizando un criterio ponderado basado en frecuencia de uso, costo de mantenimiento y valor de reposición.

El objetivo es apoyar la priorización de mantenimiento y la toma de decisiones operativas en entornos industriales.

Este repositorio se presenta como un **caso académico con datos simulados**, sin utilizar información confidencial ni real.

---

## Funcionalidades
- Ingreso de datos por consola
- Cálculo automático de clasificación ABC
- Generación de gráfico de distribución ABC
- Generación de informe en PDF
- Código portable y reutilizable

---

## Requisitos
- Python 3.9 o superior
